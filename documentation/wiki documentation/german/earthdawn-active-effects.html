Earthdawn Active Effekts

Das Earthdawn System für Foundry VTT baut stark auf Aktive Effects auf. Diese wurden erheblich angepasst um den Anforderungen des Systems gerecht zu werden.

Grundlagen des Effekt Dokumentes

Das Effektdokument besitzt vier Tabs:

  1. Allgemein Tab
  2. Dauer Tab
  3. Anpassungs Tab
  4. Script Tab

Allgemein Tab

Im Allgemein Tab können die folgenden informationen angegeben werden:

  • Effektbild
  • Effektname
  • Zustände
  • Zuweisung
  • Beschreibung

Zum Bild, dem Namen und der Beschreibung ist nicht viel zu erklären. Die Zustände sind aktuell noch eine einfach Auswahl, die besonderheit von mehrfachen Zuständen ist aktuell noch nicht umgesetzt. so gibt kann hier zur Zeit nur halbe Deckung oder teilweise Bewegungseinschränkung ausgewählt werden.

Die Zuweisung bedarf der größten Erklärung. Es ist wichtig den Unterschied zu machen ob ein Effekt für Akteur bestimmt ist oder für das Item selbst. EIn Effekt für einen Akteur muss mit den beiden Optionen

Zuweisungsfunktionen von Aktiven Effekten

Alle Fähigkeiten, Kräfte und Zauber, die einen Earthdawn Active Effect haben, erzeugen beim würfeln einen Button im Chat. Durch klicken auf diesen Button wird der Effekt den aktuell anvisierten Zielen zugewiesen.

Effekte reagieren auf extra Erfolge und erhöhen ggf. die Dauer oder Wirkung.
