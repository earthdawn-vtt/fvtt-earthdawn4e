Release Notes

Im folgend Abschnitt werden alle Funktionen des Earthdawn Systems für Foundry VTT beschrieben.

Grundlagen

Für die Entwicklung vom Earthdawn System wurde der Fokus auf Akteure, Items und Chat Nachrichten gesetzt. Token, Szenen etc. wurden teilweise mitangepasst, werden aber nicht separiert beschrieben.

Akteure

Im Earthdawn System für Foundry VTT werden aktuell zwischen neun unterschiedlichen Akteuren unterschieden.

  • Charakter
  • NSC (Nicht Spieler Charakter)
  • Kreaturen
  • Geister
  • Drachen
  • Dämonen
  • Gruppen
  • Fallen
  • Fahrzeuge

Charaktere

Charaktere sind die wichtigste Art der Akteure und haben entsprechend auch die meisten Funktionen.

Im folgenden werden die einzelnen Funktionen die ein Charakter Akteur besitzt beschrieben.

  • Das System bietet eine Charaktererschaffung. hierzu gibt es einen dedizierten Wiki Artikel
  • Der Charakterbogen besitzt automatisch berechnetete Charakteristiken (siehe Wiki Artikel Charakterbogen)
  • Attribute, Halbmagie und Allerweltsfertigkeiten können über die Attribute gewürfelt werden.
  • Talente, und Zauber können als Favoriten (Makros) auf das Allgemein-Tab gebracht werden.
  • Fähigkeiten (Talente, Weihekräfte und Fertigketen) sind in der Regel die Auslöser für Aktionsabläufe (Angriffe etc.)
  • Matrix, Grimoire und Rohe Magie stehen als Zauberoptionen zur Verfügung.
  • Ausrüstung haben einen Status der für die Benutzung und Berechnung von werten wichtig ist.
  • Ausrüstungsgewicht kann automatisch berechnet werden um kleineren (Windlinge) oder größeren (Trolle) Namensgebern zu optimieren.
  • Earthdawn Effekte können direkt am Token gesetzt werden.
