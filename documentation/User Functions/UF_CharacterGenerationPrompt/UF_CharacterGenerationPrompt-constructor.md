The Constructor is a technical collection of all required data for the Character Generation

**Technical Information:**
The constructor of the CharacterGenerationPrompt class initializes the character generation process by: <br>

1. Setting up the character generation data model.<br>
2. Loading document collections (e.g., namegivers, disciplines, questors, skills, spells, and equipment).<br>
3. Configuring available attribute points and language settings.<br>
4. Defining the steps of the character generation process.<br>
5. Establishing the initial state of the tab navigation system.<br>
