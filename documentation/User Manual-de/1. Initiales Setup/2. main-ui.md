# Chat

Im Folgenden Abschnitt werden die Erweiterungen für den Chatbereich genauer erläutert.<br>

## Chatbefehle

### Hilfe

Alle Chat Funktionen können auch im spiel mit dem Chatbefehl **/help** oder **/h** aufgerufen werden. Hierauf folgt eine Chatnachricht mit allen verfügbaren Chatbefehlen und ihren Funktionen.

### Eine Willkürliche Stufe Würfeln

Earthdawn benutzt das Stufen system zum würfeln und nicht immer hat man einen Akteur zur hand um etwas bestimmtes zu würfeln. Dafür wurde der Willkürliche Wurf eingeführt. Mittels dem **/s** kann man einfach eine bestimmte Stufe würfeln ohne weiteren Kontext (z.B. "/s 15" würde 2w6 + 1w12 Würfeln).

### Charaktererstellung

Um schnell und einfach einen neuen Charakter zu erstellen, wurde die Charaktererschaffung eingebaut. Diese lässt sich unteranderem über den Chatbefehl **/char** aufrufen. <br>
Die genaue Funktion der Charaktergenerierung ist hier zu finden [Charaktererstellung](../2.%20Akteure/Charakter.md)

### Legendenpunkte verteilen

Legendenpunkte sind ein wichtiger Teil des Earthdawn systems. Um diese als Spielleiter einfach und mit einer entsprechenden Beschreibung an die richtigen Spieler zu verteilen gibt es den **/lp** Chatbefehl. <br>
Dieser offnet ein Fenster mit meheren Optionen. Es gibt ein Feld für die Legendenpunkte und ein weiteres für eine Beschreibung. Zusätzlich gibt es noch drei Auswahlfelder um die Charaktere zu bestimmen, die diese Prämie erhalten sollen. Das erste Feld zeigt alle Charaktere die aktuell von einem angemeldeten Benutzer gespielt werden (Konfiguriert sind). Die nächste Auswahl zeigt alle Charaktere die zwar einem Benutzer zugewiesen sind, welcher aber gerade nicht aktiv ist. Die letzte Auswahl zeigt alle Charaktere auf die Spieler über Rechte zugriff haben, die jedoch nicht zugewiesen sind.<br>
Für jeden so ausgewählten Akteur wird ein neuer Eintrag in der *LP Übersicht* mit der Beschreibung und den Legendenpunkten erzeugt.
