# Chat

Im Folgenden Abschnitt werden die Erweiterungen für den Chatbereich genauer erläutert.<br>

## Chatbefehle

### Hilfe

Alle Chat Funktionen können auch im spiel mit dem Chatbefehl **/help** oder **/h** aufgerufen werden. Hierauf folgt eine Chatnachricht mit allen verfügbaren Chatbefehlen und ihren Funktionen.

### Eine Willkürliche Stufe Würfeln

Earthdawn benutzt das Stufen system zum würfeln und nicht immer hat man einen Akteur zur hand um etwas bestimmtes zu würfeln. Dafür wurde der Willkürliche Wurf eingeführt. Mittels dem **/s** kann man einfach eine bestimmte Stufe würfeln ohne weiteren Kontext (z.B. "/s 15" würde 2w6 + 1w12 Würfeln).

### Charaktererstellung

Um schnell und einfach einen neuen Charakter zu erstellen, wurde die Charaktererschaffung eingebaut. Diese lässt sich unteranderem über den Chatbefehl **/char** aufrufen. <br>
Die genaue Funktion der Charaktergenerierung ist hier zu finden [Charaktererstellung](../2.%20Akteure/Charakter.md)

### Legendenpunkte verteilen

Legendenpunkte sind ein wichtiger Teil des Earthdawn systems. Um diese als Spielleiter einfach und mit einer entsprechenden Beschreibung an die richtigen Spieler zu verteilen gibt es den **/lp** Chatbefehl. <br>
Dieser offnet ein Fenster mit meheren Optionen. Es gibt ein Feld für die Legendenpunkte und ein weiteres für eine Beschreibung. Zusätzlich gibt es noch drei Auswahlfelder um die Charaktere zu bestimmen, die diese Prämie erhalten sollen. Das erste Feld zeigt alle Charaktere die aktuell von einem angemeldeten Benutzer gespielt werden (Konfiguriert sind). Die nächste Auswahl zeigt alle Charaktere die zwar einem Benutzer zugewiesen sind, welcher aber gerade nicht aktiv ist. Die letzte Auswahl zeigt alle Charaktere auf die Spieler über Rechte zugriff haben, die jedoch nicht zugewiesen sind.<br>
Für jeden so ausgewählten Akteur wird ein neuer Eintrag in der *LP Übersicht* mit der Beschreibung und den Legendenpunkten erzeugt.

## Chatnachrichten

Die Chatnachrichten wurden für das Earthdawn system erweitert. Im folgenden Abschnitt wird auf diese Erweiterungen eingegagnen. 

### Generelles

alle Chatnachrichten weisen ein kleines Bild des würfelnden Akteurs auf. Dieses Bild kann entweder das Bild des ausgewählten Token sein, oder ein fest am Benutzter hinterlegten Bild (siehe [Chat Avatar](./3.%20System-Settings.md)). <br>
Des Weiteren gibt es in der Regel einen Text der eine kurze Information darüber gibt, welcher Akteur welche Aktion ausgeführt hat. <br>
Sollte eine Aktion ein Ziel ausgewählt haben, so ist der Token des Ziels auch in der Chatnachricht enthalten und kann weitere Klickoptionen mitsichbringen (siehe Reaktionen)

### Aktions- und Wirkungswürfe

Earthdawn unterscheidet in Aktions- und Wirkungswürfe. Der für das System relevante unterschied ist, dass ein Aktionswurf immer eine Zielschwierigkeit hat und entsprechend Erfolge erzielen kann. <br>
Die Chatnachrichten von Würfelwürfen wurden so erweitert, dass es die folgenden zusatz informationen gibt:
- Zielschwierigkeit (ein kleines Fadenkreuzsymbol) (nur bei öffentlichen würfen sichtbar)<p class="fa-light fa-crosshairs"></p>
- Würfelergebnis (ein kleines Würfelsymbol) <i class="fa-regular fa-dice"></i>
- Anzahl an Erfolgen (zwei kleine Häkchen) <i class="fa-light fa-check-double"></i>
<br>
Des Weiteren gibt es auch noch eine zusammenfassung der modifikatoren. Einmal für die Stufenmodifikatoren und einmal für die Modifikatoren der Zielschwierigkeit. Diese beidem Übersichten lassen sich aufklappen wenn das kleine Dreiecksymbol geklickt wird.<br>
Ein Mausklick auf das das große Würfelergebnis zeigt die genauen Würfelergebnisse mit eventuellen explodierten Würfeln.<br>
Zu guter letzt wurde die Textausgabe noch soweit angepasst, dass klar ist, welche Würfel zu der Stufe gehören und welche Bonuswürfel (z.b. durch Karma) zu dem Wurf hinzukommen.<br>
Für Situationen wo die Schwierigkeit bestimmt werden kann (z.b. im Kampf) berechnet das System die Schwierigkeit automatisch. Es gibt allerdings auch immer wieder Situationen während des Spiels, die kein Ziel zur Berechnung vorweisen. Solche Würfelwürfe gehen immer automatisch gegen die Mindestschwierigkeit die im System gesetzt wurde (siehe [Mindestschwerigkeit](./3.%20System-Settings.md)). Natürlich ist es auch möglich die Schwierigkeit manuell im Würfeldialog zu setzten. <br> 
Entsprechend der Schwierigkeit werden die Erfolge berechnet. Wenn die Schwierigkeit 0 beträgt werden keine Erfolge berechnet, was für die meisten Effektwürfe uninteressant ist.<br>

### Sonderaktionen im Chat

Es gibt eine Reihe von Aktionen die weitere Interaktionen hervorrufen, ein Angreifer will ggf. einem Angriff ausweichen oder es soll nach einem erfolgreichem Zauber die Wirkung ermittelt werden. Da diese Aktionen bei Earthdawn unmittelbar miteinander verknüpft sind gibt es für solche Aktionen eine Reihe von Buttons die je nach Wurftyp mit angezeigt werden. 

#### Reaktion

Wenn das Ziel einer Aktion über Reaktionsfähigkeiten verfügt ( siehe [Reaktionen](../Items/2.%20Abilities.md)), entsteht unter seinem Bild in der Chatnachricht, die Option Reaktionen auszuwählen und einen eigenen Wurf auszulösen. <br>
Aktuell werden die ursprünglichen Chatnachrichten **nicht** durch eine erfolgreiche Reaktion verändert. Die Zielschwierigkeit der Reaktion ist immer das würfelergebnis.

#### Schaden

Angriffe haben immer einen Schadenswurf zur Folge. Dies führt immer zu einem zusätzlichen Button in der Chatnachricht, die von dem Spieler der die Nachricht ausgelöst hat oder vom Gamemaster ausgelöst werden kann. <br>
auch diese Aktion erfordert ein Ziel und es ist Möglich, das Ziel zwischendurch zu wechseln. Sollte dadurch einmal der Schaden versehentlich einem Ziel zugewiesen werden, besteht auch die Möglichkeit dies wieder rückgängig zu machen. Hierfür existiert ein kleines Symbol <span class="fa-undo"></span> unter dem Button pro Ziel, das den Schaden genommen hat.
