Das Earthdawn Foundry System bringt eine große Anzahl an Einstellungsmöglichkeiten mit sich um den Spielern möglichst viel Freiheit zu geben. In folgendem Kapitel werden die einzelnen System Einstellungen beschrieben und ihre Auswirkungen erläutert.

## Charaktergenerierung

### AutoCharGen

**Type:** checkbox <br>
**Standardeinstellung:** aktiv<br>
**Beschreibung:** Wenn diese Einstellung aktiviert ist, erscheint ein kleines Fenster beim erstellen eines Akteurs vom Typ Charakter, dass den Benutzer fragt ob die Charaktererstellung genutzt werden soll oder nicht. Ist diese Einstellung jedoch deaktiviert, wird direkt ein leerer Akteur erstellt.<br>

### charGenAttributePoints

**Type:**  Zahlenfeld<br>
**Standardeinstellung:** 25<br>
**Beschreibung:** Hier werden die möglichen Attributspunkte eingetragen, die während der Charaktererstellung für die Attributswertbestimmtung benutzt werden können.<br>

### charGenMaxRank

**Type:** Zahlenfeld<br>
**Standardeinstellung:** 3<br>
**Beschreibung:** Dieses Feld bestimmt den Maximalrang eines Talents, einer Fertigkeit oder einer Weihekraft, die ein Benutzer während der Charaktererstellung auf eine der genannten Fähigkeiten verteilen kann<br>

### charGenMaxSpellCircle

**Type:** Zahlenfeld<br>
**Standardeinstellung:** 2<br>
**Beschreibung:** Diese Konfiguration schränkt die Auswahl der Zauberkreise ein, aus denen ein Benutzer während der Charaktererstellung wählen kann.<br>

## Sonstige

### chatAvatar

**Type:** Auswahlfeld <br>
**Standardeinstellung:** Bild aus der Spieler Konfiguration<br>
**Beschreibung:** Mit dieser Einstellung kann der Spielleiter festlegen, ob die im chatfenster angezeigten Charakterbilder dem Konfigurierten Bild eines Benutzers entsprechen oder dem es zugewiesenen Akteur Tokens. Für Spielleiter verhält sich die Einstellung etwas anders. Die Standardeinstellung ist wie für Spieler und bezieht sich auf das Konfigurierte Bild. bei der Auswahl "ausgewählter Token" wird jedoch der Token gewählt den der Spielleiter gerade ausgewählt hat (oder es wird auf das Konfigurationsbild zurückgegriffen)<br>
